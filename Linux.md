# Linux系统目录结构大全
```vb
linux系统目录结构
/bin:Binary的缩写，它存放着最经常使用的命令：
ps ls ping grep cp

/sbin:s是Super User的缩写，系统管理员使用的系统管理程序例如：
iptables reboot fdisk ifconfig swapon

/boot:存放启动linux时的核心文件，包括链接文件与镜像文件

/dev:是Device得到缩写，存放的是Linux外部设备，访问外部设备时与访问文件一致

/etc:存放所有系统管理需要的配置文件和子目录
也包含了用于启动停止单个程序的shell脚本

/home:用户家、主目录，Linux下每个用户都有一个自己的家目录，命名一般为用户账号命名

/lib:存放系统最基本的动态链接共享库，作用类似windows下的dll

/lost+found:一般是空的目录，当系统无法关机时，会存储一些文件

/media linux:系统会识别一些设备，例如u盘，光驱。等识别后会把设备挂载到这个目录下

/mnt:系统系统这个目录是为了让用户临时挂载别的文件系统，可以将光驱挂载到这个
/mnt/下，然后进入目录就可以查看光驱里的内容了

/opt:主机额外的安装软件所摆放的目录。

/proc:是一个虚拟目录，它是系统内存的映射，可以直接访问这个目录来获取系统信息

/root:它是超级管理员文件目录

/selinux:是Redhat/CentOs所持有的目录，它是一个安全机制，类似window的防火墙，
但是这套机制比较复杂。

/srv:存放系统服务启动后需要提取的数据

/tmp:临时文件夹

/usr:用户程序存放的目录

/usr/bin:系统用户使用的应用程序

/usr/sbin:超级用户使用的比较高级的管理程序和系统守护程序

/usr/src:内核源代码目录

/var:经常被修改的目录放在这儿，包括各种日志文件

```

```vb
切换用户：su 用户名(超级管理员切换不需要密码，当频繁切换时用使用exit不要递归循环切换)
查看文件：ls or ls -a
创建文件：touch 文件名
重命名：mv 路径原文件名 新路径文件名
复制：cp 路径文件名 新路径文件名
删除：rm -f 强制删除
查看帮助：命令 --help
创建文件夹:mkdir 文件目录名
递归创建目录：mkdir -p 文件目录/文件目录/文件目录
递归强制删除目录:rm -rf 文件目录

运行模式：
0 关机
1 单用户模式
2 多用户
3 切换到命令行模式
4 表示未被使用模式
5 切换到桌面模式
6 重启
例子： init 0

ctrl+c:命令行终止
ctrl+a:单条命令行首部
ctrl+e:命令行尾部
ctrl+l:清屏
箭头上与下：上一个命令下一个命令
Tab:命令自动补全
关机命令：poweroff
shutdown now:现在关机
shutdown now -h 加时间设置多久之后关机
重启：reboot

文件操作：
输出内容 > 文件路径/文件名
> 覆盖写入 删除之前的内容覆盖写入
>> 追加写入 不覆盖添加新的内容
举例：echo 1 > 1.txt

查看文件：
cat 文件名

shell > more filename //回车方式逐行查看文件内容 空格分页查看 
shell > less filename //上下左右键查看文件各个部分
shell > head -n filename //查看文件前n行内容
shell > tail -n filename //查看文件最末尾n行内容
shell > wc filename //计算文件行数
more less 按q退出当前文件
wc:文件行数 字符或字节数 单词数可查看 wc --help

vim编辑器：
vim 文件名
vim +line 文件名
字符集：上h下j左k右l
单词级: b w e
行级：行首^ 尾$
gg:跳到文件第一行
G：文件最后一行
M:跳到屏幕中间行
W:下一个单词
b:上一个单词

yy:复制一行
5yy:复制5行
yw：复制一个单词，5yw复制5个单词
yG：复制当前文件到下一行
y$：复制当前光标整行到当前行

p:当前位置粘贴
P:当前位置向前粘贴

dd:删除一行 3dd:删除5行
dw:删除一个单词 5dw:删除5个单词
d^:从当前光标开始删除左边行的所有
d$:删除当前行？？？

查找文件：
/文件名 or :/文件名/
?:向上找
找到之后按n N进行上一个下一个查看

u:撤销上一个动作
ctrl+r:重做上一个操作
. : 重复前一个操作

取消高亮：
:nohl

替换：
:s/查找的内容/替换的内容  //只替换光标所在位置
:s/查找的内容/替换的内容/g //替换文档所在行所有
:%s/查找的内容/查找的内容/g //全部替换

:5 :定位到第五行

临时设置行号
:set number or:set nu
取消：
:set no nonumber or :set nonu

在用户家目录设置默认项使vim默认显示行号
/etc/root/.vimrc
保存文件：
:x 保存退出
:X 加密文件 不要加密系统文件 加密再后保存 如果不想加密就在加密后再加密码一次密码设为空
:u 撤销上一次操作
Ctrl+r:取消撤销
撤销多步：数字 u

打开文件指定行号：
vim +行号 文件名;
vim +6 passwd

打开指定搜索的关键词：
vim +/keyword 文件路径
yy 复制当前行
p 粘贴

set：查看当前所有变量
set | more : 分页查看 空格换页
```
