# git 的入门操作

**在当前目录下创建一个本地仓库**

git init

**初始配置**

git config --global user.email "1@qq.com"
git config --global user.name "dandan"

**创建一个文件**

touch

**使用 vim 编辑一个文件**

vim test.php

**添加到暂存区**

git add .(代表所有文件也可以指定文件名)

**提交**

git commit -m '注释';

**查看状态**

git status

**查看分支**

git branch

**查看日志**

git log
